Тестовое задание выполнено при помощи следующих технологий:

  • FastApi
  • SQLAlchemy
  • SQLite
  • Jinja2
  • Docker
  • Uvicorn

Реализована дефолтная система CRUD для моделей Users и Secretdata. Также для Secretdata реализованы эндпоинты с ТЗ, для сбора и обрабртки зашифрованной информации.

Эндпоинт "secretdata/encrypted" делает get запрос по указанному в ТЗ адресу, достает данные и записывает из в таблицу. Возвразает список зашифрованных строк.

Эндпоинт "secretdata/decrypted" делает post запрос по указанному в ТЗ адресу, расшифровывает данные и записывает их в таблицу. Возвразает список расшифрованных строк.
